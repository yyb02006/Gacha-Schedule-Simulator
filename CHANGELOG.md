# Changelog

## [1.0.3] - 2025-11-28

### 수정

- 세부옵션 모드의 추가 재화 누계가 기본옵션 모드의 재화로 누계되는 버그 수정
- 배너별 진입 시 평균재화 차트의 툴팁에서 첫 번째 행에 추가 투입 재화를 표시하록 수정

## [1.0.2] - 2025-11-27

### 수정

- 브러쉬 이동 후 배너별 성공 기록 툴팁의 인덱스 계산 오류 수정
- 모든 배너 비활성화 후 시뮬레이션 실행 시 activePickupDatas의 길이가 0이 된 채로도 실행되는 버그 수정
- 시뮬레이션 통계 차트의 텍스트 크기 및 두께 수정
- Alert컴포넌트 재사용할 수 있도록 수정, 이제 취소 버튼의 표시 유무를 프로퍼티로 전달할 수 있음

### 제거

- 불필요한 워커 실행결과 로그 제거

## [1.0.1] - 2025-11-26

### 수정

- 세부옵션 모드에서 6성 올명함, 6성 올풀잠 버튼 적용 시 6성이 아닌 모든 오퍼레이터에게도 적용되는 버그 수정
- 웹페이지 새로고침시마다 한정 배너의 첫 번째 6성 오퍼레이터의 한정 타입이 바뀌는 버그 수정
